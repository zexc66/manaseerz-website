## Image Acquisition Strategy

### Phase 1: Immediate Actions (Week 1-2)
1. **Audit Completed Projects**
   - List all projects completed in past 12 months
   - Identify 10-20 best projects for photography
   - Contact clients for permission to photograph

2. **Hire Professional Photographer**
   - Interview 2-3 photographers specializing in interior/real estate
   - Review portfolio for lighting and composition quality
   - Request bid for package (10-20 projects)
   - Schedule photo shoots during optimal lighting conditions

3. **Create Photography Kit**
   - Provide photographer with brand guidelines (gold/black palette)
   - Share PHOTOGRAPHY_GUIDE.md with specific shot lists
   - Prepare client consent forms

### Phase 2: Professional Photography (Week 3-8)
1. **Photo Shoot Schedule** (10-20 projects)
   - Chandelier Installations: 4 projects
   - EV Charger Installations: 4 projects
   - Smart Home Projects: 4 projects
   - Renovation Projects: 4 projects
   - Outlet/Circuit Work: 2 projects
   - Range Hood Installations: 2 projects

2. **On-Site Deliverables per Project**
   - 1-2 wide shots (room context)
   - 2-3 detail shots (close-ups)
   - 1-2 before/after comparisons (renovations)
   - Optional: team at work, client testimonial

3. **Post-Processing**
   - Color correction to match brand palette
   - Exposure and contrast adjustments
   - Image optimization (WebP, multiple sizes)

### Phase 3: Image Integration (Week 9-12)
1. **Replace Unsplash Images**
   - Upload professional photos to folder structure
   - Update portfolio.tsx with real project images
   - Add alt text and metadata

2. **Create Additional Content**
   - Service page hero images
   - Before/after gallery
   - Team photos
   - Equipment/tool showcase

3. **Website Update**
   - Deploy new images to production
   - Test image loading and optimization
   - Update sitemap with new image URLs

## Photography Brief Examples

### Example 1: Crystal Chandelier (Frisco, TX)
```
Client: Private residence, Frisco, TX
Project: 12-light crystal chandelier in cathedral ceiling
Home Value: $1.2M+

Required Shots:
1. Wide shot from dining room doorway (24mm, f/8)
   - Show full ceiling height and room context
   - Include dining table for scale

2. Low angle looking up (16mm, f/5.6)
   - Emphasize cathedral ceiling
   - Capture light refraction

3. Detail of crystal elements (50mm, f/4)
   - Close-up of individual crystals
   - Show quality and craftsmanship

4. Mounting hardware (50mm, f/4)
   - Secure installation detail
   - Safety and professionalism

5. With homeowner (optional)
   - Natural interaction
   - Show scale and impact

Lighting: Warm white LEDs (3000K)
Time: Late morning (10am-12pm)
Budget: $300-400 for professional photographer
```

### Example 2: Tesla Wall Connector (Plano, TX)
```
Client: Luxury estate, Plano, TX
Project: Tesla Wall Connector installation with 48A circuit upgrade
Home Value: $2M+

Required Shots:
1. Charger on wall with Tesla Model 3 (24mm, f/8)
   - Show charger in context
   - Include vehicle for scale

2. Connection detail (50mm, f/4)
   - Plug and cable management
   - Clean, professional installation

3. Circuit panel upgrade (35mm, f/8)
   - Before: 100A panel
   - After: 200A panel with new breaker
   - Clear labeling and organization

4. Conduit installation (50mm, f/4)
   - Neat, professional routing
   - Code compliance

5. Mobile app interface (50mm, f/4, screen capture)
   - Show charging status
   - Demonstrate smart features

Location: Attached garage
Vehicle: Tesla Model 3 or Model Y
Time: Afternoon (1pm-4pm)
Budget: $350-450
```
