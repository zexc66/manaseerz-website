## Overview

This guide provides comprehensive instructions for capturing high-quality photographs of Manaseerz Electric's completed projects. All images should reflect:
- **Professionalism**: Clean, organized work areas
- **Quality**: Precise installations with attention to detail
- **Luxury Aesthetic**: High-end home environments ($500K+ properties)
- **Safety**: Proper tools, equipment, and protective gear

## Equipment Recommendations

### Minimum Setup (Beginner)
- **Camera**: Modern smartphone (iPhone 14+ or equivalent)
- **Lighting**: Natural light + small ring light
- **Tripod**: Smartphone tripod for stability
- **Cleaning**: Microfiber cloth for equipment

### Recommended Setup (Intermediate)
- **Camera**: DSLR or Mirrorless (Sony A7 series, Canon R series)
- **Lenses**: 24-70mm f/2.8, 16-35mm f/4 for wide shots
- **Lighting**: External flash + reflector
- **Tripod**: Full-size tripod with pan/tilt head

### Professional Setup (Advanced)
- **Camera**: Full-frame DSLR/Mirrorless
- **Lenses**: Prime 35mm f/1.4, 24-70mm f/2.8, 16-35mm f/2.8
- **Lighting**: Strobe lights with modifiers, continuous LEDs
- **Accessories**: Polarizer, ND filters, color checker card

## General Guidelines

### Lighting
- **Natural Light**: Shoot during golden hour (10am-2pm) for soft, warm light
- **White Balance**: Set to 3200-4000K warm white to match home interior lighting
- **Mixed Light Sources**: Avoid mixing daylight with indoor lighting (turn off overhead lights during daytime shoots)
- **Reflectors**: Use white reflector to bounce light into shadow areas
- **HDR**: Use bracketing for high-contrast scenes (3-5 exposures)

### Composition
- **Rule of Thirds**: Place key elements on grid intersections
- **Leading Lines**: Use room architecture, countertops, edges to guide viewer's eye
- **Depth of Field**: f/5.6-f/8 for overall sharpness, f/2.8 for selective focus
- **Eye Level**: Shoot at standing eye level (5'6" for interiors)
- **Include Scale**: Show room context to demonstrate project size

### Technical Settings
- **Aperture**: f/5.6-f/8 for maximum depth of field
- **ISO**: 100-400 for clean images (avoid noise)
- **Shutter Speed**: 1/125s or faster handheld, 1/60s minimum on tripod
- **File Format**: RAW (best) or high-quality JPEG
- **Resolution**: Minimum 3000x2000px (6MP)
- **White Balance**: Auto in mixed light, or set manually (3200-4000K)

### Workflow
1. **Preparation**: Clean area, organize tools, ensure all equipment is off
2. **Scout Location**: Identify best angles and lighting conditions
3. **Wide Shots**: Capture full room context first
4. **Detail Shots**: Close-ups of key installations and craftsmanship
5. **Before/After**: Document transformation when possible
6. **Team Photos**: Capture technicians working professionally
7. **Review**: Check images on-site for issues

## Editing & Optimization

### Required Edits
1. **White Balance**: Correct to warm white (3200-4000K)
2. **Exposure**: Ensure proper brightness (no blown highlights)
3. **Contrast**: Moderate contrast for depth
4. **Sharpness**: Apply subtle sharpening
5. **Noise Reduction**: Minimal (maintain detail)

### Optional Enhancements
- **Vignette**: Subtle darkening of corners to focus attention
- **Color Grading**: Warm gold tones to match brand palette
- **Lens Correction**: Remove distortion on wide-angle shots
- **HDR Merge**: For high-contrast scenes (multiple exposures)

### Export Settings
- **Format**: JPEG (80% quality) or WebP
- **Resolution**: 1000px width (portfolio), 1920px (modal)
- **Color Space**: sRGB
- **Metadata**: Include title, description, location

### File Naming Convention
```
portfolio/[category]/[project-name]-[city]-[##].jpg

Examples:
portfolio/chandelier/crystal-chandelier-frisco-01.jpg
portfolio/ev-charger/tesla-wall-connector-plano-01.jpg
portfolio/smart-home/lutron-caseta-mckinney-01.jpg
portfolio/renovation/kitchen-remodel-lewisville-01.jpg
portfolio/outdoor/usb-c-outlet-allen-01.jpg
portfolio/range-hood/stainless-hood-plano-01.jpg
```

## File Organization

### Folder Structure
```
public/
├── portfolio/
│   ├── chandelier/
│   ├── ev-charger/
│   ├── smart-home/
│   ├── renovation/
│   ├── outdoor/
│   └── range-hood/
├── services/
│   ├── hero/          # Large hero images for service pages
│   ├── detail/        # Detailed project images
│   └── thumbnails/    # Small preview images
└── team/              # Team photos and headshots
```

### Image Sizes
- **Portfolio Grid**: 800x600px (4:3 ratio)
- **Portfolio Modal**: 1920x1080px (16:9 ratio)
- **Service Hero**: 1600x900px (16:9 ratio)
- **Service Detail**: 800x600px (4:3 ratio)
- **Thumbnails**: 400x300px (4:3 ratio)
- **Mobile Optimized**: 600x400px

### Image Optimization Tools
- **TinyPNG**: https://tinypng.com/ (free online compressor)
- **Squoosh**: https://squoosh.app/ (Google's image optimizer)
- **ImageOptim**: Mac app for lossless compression
- **Adobe Lightroom**: Professional editing and export

## Common Mistakes to Avoid

1. **Poor Lighting**: Dark or overly bright images
2. **Cluttered Background**: Distracting elements in frame
3. **Bad Angles**: Unflattering camera positions
4. **Motion Blur**: Blurry images from camera shake
5. **Wrong White Balance**: Unnatural color casts
6. **Missing Context**: Not showing room/scale
7. **Incomplete Shots**: Missing key installation details
8. **Safety Violations**: Visible hazards or improper work

## Timeline & Workflow

### On-Site (2-3 hours)
1. **Preparation** (15 min): Clean area, organize equipment
2. **Wide Shots** (30 min): Capture full room context
3. **Detail Shots** (45 min): Close-ups of key installations
4. **Team Photos** (15 min): Technicians working (optional)
5. **Review** (15 min): Check images for issues

### Post-Processing (1-2 hours)
1. **Culling**: Select best images
2. **Editing**: Color correct, exposure, sharpening
3. **Export**: Optimize and save in required formats
4. **Organization**: Organize in folder structure
5. **Upload**: Add to website/portfolio
